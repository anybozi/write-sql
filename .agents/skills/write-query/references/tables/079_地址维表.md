# 地址维表

- **Hive 表名**: `zone_gz_yz.dwd_yz_addr_final`
- **用途**: 通过主业务表中的地址标识补充中文地址名称，或由装机标准地址回填指定层级地址 ID/名称。

---

## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| id | decimal(24,0) | 地址id |
| addr | string | 地址 |
| grade | int | 级别 |
| parentid | decimal(24,0) | 上级 |
| addr_id_6 | decimal(24,0) | 6级地址id |
| addr_id_7 | decimal(24,0) | 7级地址id |

---

## 本表常用条件速查（A 层附注）

| 叫法/场景 | 条件片段 |
|----------|---------|
| 装机地址中文名 | `grade = 10` |
| 069 全业务资料表补地址 | `main.serv_addr_id = CAST(addr_dim.id AS STRING)` |
| 069 补指定地址层级 | `main.serv_addr_id = CAST(addr_dim.id AS STRING)` 后按目标 `grade` 经 `parentid` 或 `addr_id_*` 上卷 |
| 069 补 9 级地址 | `serv_addr_id -> parentid -> 9级地址`，最终关联加 `grade=9` |
| 069 补 7 级地址 | `serv_addr_id -> addr_id_7 -> 7级地址`，最终关联加 `grade=7` |
| 补 5 级地址 | `serv_addr_id -> addr_id_6 -> 6级地址.parentid -> 5级地址`，最终关联加 `grade=5` |

> 地址 ID 类型提醒：069 `serv_addr_id` 是字符型，本表 `id/addr_id_7` 是 `decimal(24,0)`。关联时统一转字符，禁止默认 `CAST(main.serv_addr_id AS DECIMAL(24,0)) = addr_dim.id`，避免长地址 ID 或异常字符转换失败导致漏数。

## 常用 SQL 片段

```sql
LEFT JOIN (
    SELECT DISTINCT
        id,
        addr
    FROM zone_gz_yz.dwd_yz_addr_final
    WHERE grade = 10
) addr_dim
    ON main.serv_addr_id = CAST(addr_dim.id AS STRING)
```

## 指定层级地址 SQL 片段

目标层级由用户需求给出。没有目标层级时先确认，不要默认猜。

### 9 级地址示例

```sql
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr10
    ON main.serv_addr_id = CAST(addr10.id AS STRING)
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr9
    ON CAST(addr10.parentid AS STRING) = CAST(addr9.id AS STRING)
   AND addr9.grade = 9
```

### 7 级地址示例

```sql
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr_dim
    ON main.serv_addr_id = CAST(addr_dim.id AS STRING)
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr7
    ON CAST(addr_dim.addr_id_7 AS STRING) = CAST(addr7.id AS STRING)
   AND addr7.grade = 7
```

### 5 级地址示例

```sql
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr10
    ON main.serv_addr_id = CAST(addr10.id AS STRING)
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr6
    ON CAST(addr10.addr_id_6 AS STRING) = CAST(addr6.id AS STRING)
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr5
    ON CAST(addr6.parentid AS STRING) = CAST(addr5.id AS STRING)
   AND addr5.grade = 5
```

## 地址脱敏模板

```sql
CASE
    WHEN LENGTH(addr_dim.addr) < 4 THEN addr_dim.addr
    WHEN LENGTH(addr_dim.addr) = 4 THEN CONCAT(SUBSTR(addr_dim.addr, 1, 1), '*')
    WHEN LENGTH(addr_dim.addr) > 4 THEN CONCAT(SUBSTR(addr_dim.addr, 1, LENGTH(addr_dim.addr) - 4), '****')
    ELSE NULL
END AS tm_addr_name
```
