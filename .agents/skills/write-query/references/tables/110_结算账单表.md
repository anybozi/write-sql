# 结算账单表

- **Hive 表名**: `dws_tpss_jszx.dws_settle_bill`
- **业务定位**: 结算账单 / 报账 / 合同 / 合作伙伴 / 网点 / 经营主体相关事实表
- **常用过滤**: 广州分片常用 `shard='200'`
- **账期字段**: `billing_cycle_id`（结算账期，常见形态为日型账期；关联月表时通常取 `substr(billing_cycle_id,1,6)`）

## 使用说明

- 市场化承包合同下查有效揽装人时，本表只作为合同、账期、网点关系来源：`contractno`、`contract_name`、`billing_cycle_id`、`channel_id`、`channel_code`、`channel_name`。
- 本表是结算账单表，不是普通号码、订单、积分或科目收入明细表。
- 关联月维表时，通常用 `substr(billing_cycle_id,1,6)` 对齐 `par_month_id`。
