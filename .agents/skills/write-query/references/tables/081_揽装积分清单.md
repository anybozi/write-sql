## 使用边界

- 本表是揽装积分专项口径/派生清单，不作为积分类全量底表。
- 只有用户明确点名“揽装积分清单专项口径”、`ads_yz_lyf_lz` 或对应视图时，才优先使用本表。
- 用户只是要积分类数据、积分明细、号码/客户/网格/竣工时间/价值积分/激励积分等全量明细时，默认先走 012 `ads_yz_score_all_list`。
