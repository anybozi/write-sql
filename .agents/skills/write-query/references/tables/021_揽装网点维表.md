## 使用说明

- 本表由揽装人维表和网点维表关联生成，只包含有效揽装人对应的有效网点。
- `sales_code` 不唯一，揽装人关联和去重应优先使用 `staff_id`。
- 需要判断网点是否有效时看 112 网点维表；需要判断揽装人是否有效时看 111 揽装人维表。
