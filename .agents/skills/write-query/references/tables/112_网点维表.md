# 网点维表

- **日表 Hive 表名**: `zone_gz_yz.dwd_yz_sale_outlers_final`
- **月表 Hive 表名**: `zone_gz_yz.dwd_yz_sale_outlers_mon_final`
- **业务定位**: 网点维表；日表数据唯一，月表按账期快照。
- **分区字段（月表）**: `par_month_id`

---

## 关键字段

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| channel_id |  | 网点 ID |
| channel_nbr |  | 网点编码 |
| channel_name |  | 网点名称 |
| status_cd |  | 状态；`S0X` 表示无效 |
| own_operators_nbr |  | 经营主体编码 |
| own_operators_name |  | 经营主体名称 |
| subst_id |  | 网点所属区县标识 |
| subst_name |  | 网点所属区县名称 |
| branch_id |  | 网点所属营服标识 |
| branch_name |  | 网点所属营服名称 |
| par_month_id |  | 月表账期 |

---

## 使用说明

- 需要判断网点是否有效时查本表，`status_cd='S0X'` 表示无效。
- 历史账期用月表 `_mon_final`，并按 `par_month_id` 过滤；当前唯一维表可用日表。
- 查有效网点下有效揽装人时，优先用 113 揽装所属表；若 113 无记录，再回查本网点表判断网点是否无效或是否有效但无揽装人。
