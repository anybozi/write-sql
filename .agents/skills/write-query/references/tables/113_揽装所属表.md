# 揽装所属表

- **日表 Hive 表名**: `zone_gz_yz.dwd_yz_sales_man_outlers_final`
- **月表 Hive 表名**: `zone_gz_yz.dwd_yz_sales_man_outlers_mon_final`
- **业务定位**: 揽装人维表和网点维表关联生成的有效揽装人对应有效网点维表。
- **分区字段（月表）**: `par_month_id`

---

## 关键字段

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| staff_id |  | 揽装人 / 员工唯一关联键 |
| sales_code |  | 销售编码 / 揽装编码（不唯一） |
| sales_man_nbr |  | 销售员编码 |
| sales_man_name |  | 销售员姓名 |
| sales_man_post_desc |  | 销售员岗位 |
| cert_no |  | 身份证号 |
| channel_id |  | 网点 ID |
| channel_nbr |  | 网点编码 |
| channel_name |  | 网点名称 |
| own_operators_nbr |  | 经营主体编码 |
| own_operators_name |  | 经营主体名称 |
| par_month_id |  | 月表账期 |

---

## 使用说明

- 本表只包含“有效网点 + 有效揽装人”的对应关系；只有有效网点和有效揽装人才会出现在本表里。
- 查询合同网点下有效销售人员、实际工号数量、有效网点对应揽装人时，优先用本表。
- 统计实际工号数量时优先 `count(distinct staff_id)`；不要用 `sales_code` 去重。
- 若网点在本表无记录，不要直接判定网点不存在；应回查 112 网点维表和 111 揽装人维表区分：无效网点、有效网点但无揽装人、有效网点但揽装人无效。
